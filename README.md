# Fruit Catch Game

A simple, lightweight arcade game built using HTML5 Canvas and vanilla JavaScript. Players move a basket horizontally to catch falling fruits and avoid hazardous bombs before the timer runs out.

## Features

- Random selection between 7 different fruit types and bomb hazards.
- The canvas automatically scales to full screen on window resize and adjusts the basket positioning safely.
- Tracks current score, displays lives using visual heart icons, and implements a 30-second countdown timer.
- Styled to use cleanly organized assets localized within an images directory.
